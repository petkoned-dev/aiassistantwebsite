GoHighLevel Funnel Setup Guide

Follow these 4 steps to publish your AI Assistant website using the GoHighLevel funnel builder.

1

Create Your Funnel Page in GoHighLevel

Set up a blank funnel page that will hold your AI Assistant website. All brand images are already hosted on our CDN — no uploads needed.

  1. In your GHL dashboard go to Sites → Funnels.
  2. Create a new funnel (or open an existing one) and add a new step/page.
  3. Choose a blank template — we'll replace all content with custom code.
  4. Open the page in the funnel builder editor so you're ready for the next steps.
Brand images are pre-hosted. The HTML code already references these CDN URLs — no need to upload or replace anything:

2

Add Header Code (Fonts + Tailwind)

This loads the fonts, colors, and Tailwind framework your site needs. Paste this into your funnel page's header tracking code section.

Where to find it in GoHighLevel:

Option A — Page-level (recommended):
In the funnel builder → click the Settings gear icon (top-right) → scroll to "Custom Code" → paste into the "Header Code" box.

Option B — Funnel-level:
Sites → Funnels → select your funnel → Settings tab → "Head Tracking Code" → paste there. This applies to all pages in the funnel.

3

Add the CSS Stylesheet

This is all the styling — animations, hover effects, gradients, the floating orb, modal popup, etc. GoHighLevel doesn't have a dedicated CSS section, so you'll wrap it in <style> tags.

Where to paste it in GoHighLevel:

Use the "Copy With <style> Tags" button below, then paste into the same Header Code box from Step 2 — just add it after the code you already pasted.

Same location: funnel builder → Settings (gear icon) → "Custom Code" → "Header Code" box.
Important: Use the "Copy With <style> Tags" button — GoHighLevel needs the CSS wrapped in <style> tags to work. The "Copy CSS Only" option is for platforms with a dedicated CSS section.

You're Done!

Click "Preview" in the funnel builder to see your page live. Everything should look exactly like the original design — all animations, hover effects, and the popup modal included.

Troubleshooting tips:
• Page looks unstyled (plain text, no colors) — double-check that Steps 2 & 3 (Header Code) were saved correctly.
• Content looks squished or has padding — set the section/row to full width and remove default padding in the GHL builder settings.
• Animations not working — make sure the CSS from Step 3 was wrapped in <style> tags before pasting into the Header Code box.
• Images not loading — the CDN URLs should work immediately; check your browser console for errors.
